# Dependencies

```
- Node v14.15.1 (LTS) or more recent. While older versions can work it is advisable to keep node to latest LTS version

- Angular v8.2.14 as the FrontEnd development framework.

- Express v4.16.4 for the creation of the API Server.

- npm 6.14.8 (LTS) or more recent, Yarn can work but was not tested for this project

- AWS CLI v2, v1 can work but was not tested for this project

- A RDS database running Postgres (might need to be reloaded because of ever-changing udacity credentials).

- A S3 bucket for hosting uploaded pictures.

```
